Detailed Description

Entry point for the Lux compiler/interpreter build process.

This program parses command-line arguments, sets up a memory arena, and runs the build process on a provided source file.

Responsibilities

  • Validates command-line arguments.
  • Parses build configuration options.
  • Initializes the arena allocator for fast memory management.
  • Executes the build process for the provided source file.

Usage

lux build <source_file>

Example:

lux build hello.lx
